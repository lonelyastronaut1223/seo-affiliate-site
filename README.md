# SEO Affiliate 自动内容网站

## 项目简介
这是一个 SEO Affiliate 自动内容网站，旨在通过高质量的产品推荐和比较来实现盈利。该网站将利用搜索引擎优化（SEO）技术吸引流量，并通过联盟营销实现变现。

## 项目结构
- `index.html`：网站首页，包含关于我们、产品推荐、博客和联系我们的部分。
- `style.css`：网站的基本样式文件。

## 部署说明
1. 确保您已安装 [Node.js](https://nodejs.org/) 和 [npm](https://www.npmjs.com/)。
2. 克隆此项目到本地：
   ```bash
   git clone <项目的Git仓库地址>
   cd SEO_Affiliate_Automatic_Content_Site
   ```
3. 安装依赖：
   ```bash
   npm install
   ```
4. 启动开发服务器：
   ```bash
   npm run dev
   ```
5. 打开浏览器，访问 `http://localhost:3000` 查看网站。
