# SEO Content Generation Rules

## Goal
- Generate high-quality SEO content for affiliate websites.
- Content must rank in Google and convert users.
- Focus on usefulness, clarity, and trustworthiness.

---

## Content Strategy Rules

1. Content must be written for humans first, SEO second.
2. Avoid AI-like wording, repetition, and over-optimization.
3. Each page must have a clear search intent:
   - Informational
   - Commercial
   - Comparison
   - Buyer guide
4. Do NOT generate thin content.
5. Each article should provide real value and decision help.

---

## SEO Structure Rules

1. Every page must include:
   - One H1
   - Logical H2 / H3 structure
   - Clear intro (who this is for)
   - Conclusion or recommendation

2. Use natural keyword placement:
   - Title
   - First 100 words
   - One or two subheadings
   - Never stuff keywords

3. Use semantic variations instead of repeating the same keyword.

4. Prefer:
   - Bullet points
   - Tables
   - Short paragraphs
   - Clear sections

---

## Affiliate Content Rules

1. No fake claims or exaggerated marketing.
2. Be neutral and honest.
3. Always include:
   - Pros
   - Cons
   - Who should buy
   - Who should NOT buy
4. Affiliate links must be contextual and helpful.
5. Never push users aggressively to buy.

---

## Comparison Page Rules

For comparison pages (A vs B):

1. Include:
   - Spec comparison table
   - Real-world usage differences
   - Price/value discussion
2. Mention strengths of BOTH products.
3. End with:
   - Best for X users
   - Best for Y users

---

## Writing Style Rules

1. Natural, human tone.
2. No marketing fluff.
3. No AI self-references.
4. No emojis.
5. Short sentences preferred.
6. Active voice preferred.

---

## SEO Safety Rules

1. Avoid duplicate content across pages.
2. Avoid keyword stuffing.
3. Avoid copied manufacturer descriptions.
4. Avoid auto-generated looking text.
5. Content must be original.

---

## Output Rules

1. Always produce ready-to-publish content.
2. Use clean Markdown or HTML.
3. Do not explain what you are doing unless asked.
4. If information is uncertain, say so clearly.

---

## Language Rules

- Default language: English
- German version only when explicitly requested
- Keep tone neutral and professional
